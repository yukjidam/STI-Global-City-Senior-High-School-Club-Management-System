# STI SHS Club Management System with Recommendation Feature

## 📌 Project Description
This web application is designed to automate and improve the club registration process for STI Senior High School students. It also provides a structured management system for club advisers, allowing them to efficiently handle club-related activities such as member management, approvals, and monitoring.

In addition, the system includes a recommendation feature that suggests suitable clubs to students based on their interests using **TF-IDF (Term Frequency–Inverse Document Frequency)** and **Cosine Similarity** algorithms.

The system also features real-time communication and a social networking module to improve interaction between students across different clubs.

---

## 🎯 Objectives
- Automate the traditional manual club registration process  
- Provide an organized platform for club management by advisers  
- Help students discover clubs that match their interests  
- Improve communication and collaboration among students  
- Enhance engagement through social and interactive features  
- Reduce paperwork and manual validation processes  

---

## ✨ Features

### 👨‍🎓 Student Features
- Account registration and login  
- Browse available clubs  
- Apply for club membership  
- View application status  
- Receive personalized club recommendations  
- Real-time chat system with other students and club members  
- Social networking (SNS-style module for posts, interactions, and club updates)  

### 👩‍🏫 Adviser Features
- Manage assigned clubs  
- View and approve/reject student applications  
- Monitor club members  
- Update club information and requirements  
- Handle audition/tryout-based club entries  

### 🧠 Recommendation System
- Uses **TF-IDF** to analyze student input and club descriptions  
- Uses **Cosine Similarity** to match student interests with clubs  
- Provides personalized club suggestions  

### 💬 Real-Time Communication System
- Instant messaging between students  
- Club-based group chats  
- Supports real-time updates and interaction  
- Enhances collaboration and coordination within clubs  

### 🌐 Social Networking (SNS Module)
- Students can create posts and updates  
- Like and comment system for engagement  
- Club announcements and activity sharing  
- Encourages interaction between different clubs and students

## ⚙️ How the System Works
1. Students create an account and log in  
2. They fill in their interests or profile details  
3. The system processes input using TF-IDF  
4. Cosine similarity is used to compare interests with club data  
5. Recommended clubs are displayed to the student  
6. Students can apply directly to clubs  
7. Real-time chat allows communication between users  
8. SNS module enables students to post updates and interact socially  
9. Advisers manage applications and club data through their dashboard
